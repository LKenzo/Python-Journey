## Concepts

print()
Display output.

input()
Gets input from user.

Variables
Store values.

...

if
Compare values to makes decisions.

elif
Compare values after an if statement returns false.

else
Provide default action if no conditions are met.

or
Ask if either condition is true.

not equal (!=)
Compare two values if they are not equal.

and
Ask if both conditions are true.

modulo (%)
Check the remainder of a value.

bool
Data type that represent True or False as values.

pythonic expression
Condendse code and keeping it readable. 

match
compares a value against one or more case.
