## Concepts
SyntaxError
Ecountering code that violates python grammatical rules and structural design.

try
To enclosed code that might cause an error.

except 
To enclosed code that executes if an error happened.

ValueError
Built-in exception that operates when an argument recieves an inappropriate values.

NameError 
An error that occurs when a variable, function, or module name that does not exist in the current scope.

else for try and except
To enclosed code that executes if there are no errors.n

pass
A statement that tells the program to do nothing and silently ignore an error.

raise
Used to intentionally trigger an exception or error condition.
