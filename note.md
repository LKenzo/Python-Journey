## Concepts
while
Used to repeatedly execute a block of code while the condition is True.

for
Used to repeatedly execute a block of code by iterating over a sequence or collection.

list
A variable that allows you to store multiple values (collection of items). 

dict
A data structure that allows you to associate with one value with another.

None
represents the absent of a value (Null).
