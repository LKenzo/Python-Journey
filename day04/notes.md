## Concepts
Libraries
File of codes that other people has written that we can use in our code.

modules
basically a library that typically have 1 or more fucntions (To reuse the code).

random (library)
A library that has 1 o rmore functions to do things randomly.

import
Keyword in python that allows us to import the content or function from a module.

from
Keyword in python allows us to be more specific with import (Picking a specific function from that module).

statisics (library)
A library that contains function that will help analyzing a data set.

command-line arguments
Allows user to provide arguments that are passed as input to the script when running from a terminal. 

sys (library)
Contains functions that focus on the systmes (The command that the user are typing).

slices
Take a slice or a subset from a list.

packages
Third-party library that can be install.

pip
Is a program that allows us to install packages into our own machine.

APIs
A protocol that allows other software to communicate and share data with each other.

requests (Library)
Allows us to make web or internet request using python

JSON
Language agnostic format for exchanging data between computers. (JSON = "{...}")

__name__
Special variable whose value is automatically set by Python, to determine the execution context of a script.
